# NetBin

Sistema IoT de caneca inteligente con clasificación de residuos por IA y recompensas para usuarios.

## Servicios

- **broker/** — MQTT broker (Eclipse Mosquitto)
- **server/** — API REST (Spring Boot 3.3 + PostgreSQL)
- **web/** — Dashboard web (React 18 + Vite)
- **mobile/** — App móvil (React Native + Expo)
- **hardware/** — Código Raspberry Pi (Python)
